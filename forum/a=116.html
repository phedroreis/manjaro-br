Rolagem vertical no Openbox: como ativar?

ID do artigo: 116
Adaptado por: pagani
Adaptado em: 13 Abr 2014, 20:56
Descrição do artigo: O texto mostrará como ativa a rolagem vertical no Openbox.
Link externo para este artigo:
kb.html?a=116


A barra de rolagem vertical, em si tratando de um notebook, é uma maravilha. Você simplesmente desliza o dedo no touchpad e pronto. Infelizmente, no Openbox, não existe uma ferramenta gráfica para ligar essa função e se você, assim como, eu tentar instalar o pacote "xfce4-settings" para realizar tal procedimento pode ser que não funcione. Pelo menos aqui não funcionou! Entretanto, veja como é simples ativar essa função do touchpad:

Abra o arquivo de configuração localizado em /etc/X11/xorg.conf.d/50-synaptics.conf.

Depois acrescente na seção Section "InputClass" o seguinte parâmetro:

Option "VertEdgeScroll" "on".

Veja como deve ficar:

Código: Selecionar todos
# leafpad /etc/X11/xorg.conf.d/50-synaptics.conf
Section "InputClass"
        Identifier "touchpad catchall"
        Driver "synaptics"
        MatchIsTouchpad "on"
        Option "TapButton1" "1"
        Option "TapButton2" "2"
        Option "TapButton3" "3"
        Option "VertEdgeScroll" "on"


Para mais informações, você pode visitar: Arch-wiki.
