Conhecendo o Manjaro HardWare Detection (mhwd)

ID do artigo: 74
Adaptado por: Holmes
Adaptado em: 27 Dez 2012, 13:32
Descrição do artigo: O texto abaixo vai discutir o comando mhwd, conhecido como Manjaro HardWare Detection.
Link externo para este artigo:
kb.html?a=74


O Manjaro tem um comando que é super importante saber e ele é o Manjaro HardWare Detection (mhwd), que é uma característica única da nosso distro. No Wiki oficial tem um artigo que trata exatamente desse comando e abaixo foi traduzido algumas coisas desse artigo.

===============

Atualmente, existem dois tipos de comando mhwd:

1. mhwd: Permite a deteção automática e configuração de hardware de computador, que o sistema está sendo executado no momento. Isso inclui tanto hardware conectado internamente via PCI (por exemplo, placas de vídeo) e conectado externamente via USB (por exemplo, pendrive).
Obs.: O comando de mhwd ainda está em desenvolvimento, e neste momento é apenas capaz de instalar drivers para placas de vídeo conectado internamente via PCI.

2. mhwd-kernel: Permite a instalação e gerenciamento fácil de múltiplos Kernels em seu sistema.

Os comandos acima são executados automaticamente durante o processo de instalação do Manjaro, sem a necessidade de identificar e instalar os drivers necessários manualmente ou editar manualmente os arquivos de configuração. Eles também são utilizáveis através do terminal após a instalação, as características do comando mhwd incluem:

- A escolha drive livre (ou seja, open-source) ou não livre (ou seja, proprietários).
- Listagem (geral ou detalhada) e a identificação do seu hardware de sistemas.
- Identificação e listagem (geral ou detalhada) dos drivers instalados.
- Listagem de drivers disponíveis para instalação (open-source ou proprietários).
- Fácil remoção e instalação de drivers.

Todos os comandos de mhwd são realizados usando o terminal.

Código: Selecionar todos
mhwd [option(s)] <config(s)>


A sintaxe de um comando de mhwd é:

--pci = listar somente os dispositivos e driver pci
--usb = listar somente os dispositivos e driver usb
-h/--help = mostra ajuda
-f/--force = força a reinstalação
-d/--detail = mostrar informação detalhada -l/-li/-lh
-l/--list = configurações de lista disponíveis para dispositivos
-la/--listall = listar todas as configurações de driver
-li/--listinstalled = listar todas as configurações de driver instalados
-lh/--listhardware = listar as informações de hardware
-i/--install <usb/pci> <config(s)> = instalar configurações de driver
-ic/--installcustom <usb/pci> <path> = instalar a configurações personalizadas
-r/--remove <usb/pci> <config(s)> = remover configurações de driver
-a/--auto <usb/pci> <free/nonfree> <classid> = auto instalar configurações
--pmcachedir <path> = caminho do cache do gerenciador de pacotes
--pmconfig <path> = defina a configuração do gerenciador de pacotes
--pmroot <path> = defina a configuração do gerenciador de pacotes como root

Listando informações de Hardware

Para identificar e listar seu hardware de computadores, a sintaxe é:

Código: Selecionar todos
mhwd -lh [opcional: ver detalhes] [opcional: somente dispositivos PCI ou USB]


Por exemplo, uma lista detalhada de seu hardware pode ser obtida através da entrada:

Código: Selecionar todos
mhwd-lh-d


Também é possível filtrar a lista por dispositivos conectados via PCI ou USB. Neste caso, uma lista detalhada será gerada apenas de hardware com uma conexão PCI:

Código: Selecionar todos
mhwd-lh-d - pci


Listando informações do driver instalado

Para identificar e listar os drivers instalados, a sintaxe é:

Código: Selecionar todos
mhwd-li [opcional: ver detalhes] [opcional: PCI ou dispositivos USB apenas]


Por exemplo, uma lista detalhada de seus drivers instalados pode ser obtida através da entrada:

Código: Selecionar todos
mhwd-li-d


Também é possível filtrar a lista de drivers instalados em hardware conectado via PCI ou USB. Neste caso, uma lista detalhada será gerada apenas para os drivers instalados no hardware utilizados com uma conexão PCI:

Código: Selecionar todos
mhwd-li-d - pci


Listando Drivers disponíveis

Para listar todos os drivers que estão disponíveis, a sintaxe básica é:

Código: Selecionar todos
mhwd-la [opcional: PCI ou USB]


Por exemplo, uma lista de todos os drivers disponíveis para apenas dispositivos USB (e não apenas aqueles ligados ao seu sistema) pode ser obtida através da entrada:

Código: Selecionar todos
mhwd-la-usb


Para listar apenas os drivers adequados que estão disponíveis para o seu sistema, a sintaxe básica é:

Código: Selecionar todos
mhwd-l [opcional: PCI ou USB]


Por exemplo, uma lista de todos os drivers disponíveis especificamente para dispositivos conectados via PCI em seu sistema pode ser obtida através da entrada

Código: Selecionar todos
mhwd-l - pci
