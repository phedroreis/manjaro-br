Instalando o Mint Menu no Manjaro MATE

ID do artigo: 72
Adaptado por: Holmes
Adaptado em: 27 Dez 2012, 12:23
Descrição do artigo: Neste texto vamos ver como podemos instalar o Mint Menu no Manjaro com o ambiente gráfico MATE.
Link externo para este artigo:
kb.html?a=72
Infelizmente, talvez o recurso mais popular e distintivo do desktop MATE - o chamado Mint Menu- não é pré-instalados por padrão. Os usuários encontrarão um menu típico, que é totalmente funcional. Como não é possível instalar o Mint Menu no repositório oficial, ele está no repositório AUR do Arch e pode ser instalado no Manjaro com MATE. Para instalar o Mint Menu - e supondo que foi instalado todos os pacotes necessários para acessar o AUR - digite o seguinte comando no terminal:

Código: Selecionar todos
yaourt mintmenu


Veja abaixo a imagem desse processo:



Aí vamos escolher a opção número 2, que é aur/mintmenu 5.3.8-1. Uma vez que o Mint Menu foi instalado, será necessário encerrar a sessão e após o login vamos adicionar o Mint Menu assim:

1 = clique no menu iniciar para abrir as opções disponíveis.
2 = selecione Adicionar ao painel...
3 = localize o MintMenu ou digite MintMenu na barra de pesquisa.
4 = marque o MintMenu e, em seguida, clique no botão Adicionar.
5 = opcionalmente, remova o antigo menu e/ou mova o Mint Menu para a posição desejada clicando sobre eles.

Pronto, agora já temos o Mint Menu!
