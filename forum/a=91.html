Modos de identificar o seu teclado

ID do artigo: 91
Adaptado por: lobocode
Adaptado em: 27 Mai 2013, 21:56
Descrição do artigo: O texto abaixo mostrará formas diferentes para identificarmos o nosso teclado.
Link externo para este artigo:
kb.html?a=91


Um caso verídico do problema:

Uma amiga instalou o Manjaro Openbox em seu notebook e de cara teve problema com a configuração do layout do teclado. Alias, não só isso mas também para identificar qual tipo, modelo do teclado ela usa. Buscou-se soluções em fóruns diversos e basicamente o que foi relatado, foram edições de arquivos tendo em vista que você já "conheça, já saiba" qual layout/modelo você tem. Mas a pergunta que fica é: Se você já não sabe, como saberá? Pensando nisso construí-se três maneiras que vão lhe ajudar a identificar o modelo/layout do teclado, que são:

Primeira dica:

Caso você esteja vindo do Ubuntu/Debian/Derivados/Other's para o Arch/Manjaro/Chakra/Derivados, o mais seguro é que antes de você instalar o novo sistema, verificar qual modelo/layout você está usando na distribuição em que seu teclado funciona perfeitamente. Para tal, basta digitar no terminal o seguinte comando: xprop -root | grep XKB. Este comando retornará algo parecido com isto: _XKB_RULES_NAMES(STRING) = "evdev", "pc105", "br", "", ""

Segunda dica:

Outra dica, é você tentar printar na tela o modelo/layout do seu teclado através do comando setxkbmap -print -verbose 10 que provavelmente irá mostrar dados como modelo/layout e outros. Exemplo:

http://bpaste.net/show/iatw9evBXhRq43o5mhma/

Terceira dica:

Caso você não consiga encontrar o Layout do teclado e não tem nenhum sistema que possa verificar isto ( ou seja, algum sistema em que seu teclado funcionava ), você pode optar pelo comando hwinfo |grep keyboard para lhe informar o modelo do teclado. Assim, será mais fácil de encontrar ajuda, suporte para o mesmo.

Espera-se que seja de grande ajuda. Pois, há casos que fica difícil configurar alguns modelos de teclados, principalmente de aparelho móvel/mobile.
