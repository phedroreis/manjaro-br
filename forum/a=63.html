Criar um LiveUSB (pendrive/cartão) do Manjaro

ID do artigo: 63
Adaptado por: Holmes
Adaptado em: 17 Nov 2012, 16:04
Descrição do artigo: Neste texto vamos ver como instalar o Manjaro no pendrive/cartão.
Link externo para este artigo:
kb.html?a=63
As ISOs do Manjaro não suportam alguns aplicativos que geram o LiveUSB, por exemplo, o UNetbootin e o LinuxLive USB Creator são uns desses aplicativos. Diante disto, a solução para aqueles usuários de outras distros e que possuem alguma habilidade com o manuseio dos comandos é usar o dd. Vejam como usar:

Código: Selecionar todos
sudo dd if=nome.iso of=/dev/USB bs=4


a- nome.iso, é o próprio nome da iso baixada, por exemplo, manjaro-kde-0.8.2-i686.iso.
b- local, é a partição que diz respeito ao USB, geralmente é sdb.

Rodando o comando acima vai instalar o Manjaro no pendrive/cartão. Aguarde todo o processo e depois é só configurar a bios do PC para dar o boot via dispositivo USB.

Essa é uma solução geral para qualquer distribuição GNU/Linux, pois todas rodam o comando dd. Existe um aplicativo que é recomendado por um dos líderes do Manjaro, o Philip Müller, o SUSE Studio Image Writer. Todavia, esse aplicativo não está para todas as distros, ou seja, são poucas as distros que usam esse aplicativo!
