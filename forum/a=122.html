Olha aí...com pacman-mirrors manipulamos nossos mirrors

ID do artigo: 122
Adaptado por: Holmes
Adaptado em: 15 Mai 2015, 20:05
Descrição do artigo: O texto procura mostra como podemos usar o pacman-mirrors para escolhermos nossos servidores.
Link externo para este artigo:
kb.html?a=122


Como podemos escolher manualmente os nossos servidores no Manjaro? Com o comando abaixo podemos fazer isto facilmente!

Código: Selecionar todos
sudo pacman-mirrors -i -c all


Este comando irá escanear os servidores do Manjaro e no final do processo apresentará os tempos de conexão de cada mirror.







Na primeira imagem podemos notar o escaneamento, já na segunda e na terceira está o pacman-mirrors que permitirá ao usuário escolher os melhores servidores por tempo e localidade.











Nas imagens acima o usuário poderá escolher os mirrors de acordo com as suas necessidades - tempo e localidade. Para avançar de uma tela a outra é só clicar em Next page e seguir escolhendo. Após a finalização das escolhas dos servidores é só clicar em Show custom list para ver os mirrors escolhidos.





As imagens acima mostram os mirrors escolhidos neste processo! Para finalizar e sair do pacman-mirror é só clicar em Done. A imagem abaixo mostra que ocorreu Generated and saved '/etc/pacman.d/mirrorlist' Custom list, isto é, a nossa escolha foi criada e salva em /etc/pacman.d/mirrorlist.
