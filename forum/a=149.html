Volume maior que 100%

ID do artigo: 149
Adaptado por: Elppans Dark Elven
Adaptado em: 30 Out 2016, 10:39
Descrição do artigo: Como configurar o volume do sistema para aumentar o volume ao máximo apenas usando aplicativos nativos da própria distro.
Link externo para este artigo:
kb.html?a=149
Olá pessoal,

Eu gosto de simplificar algumas configurações e tenho uma dica bem simples de algo que estive procurando faz MUITO tempo.

Eu uso caixas de som que não são muito boas e portanto o som somente em 100% é muito baixo e muitas pessoas tem o mesmo problema, pelo que pude ver ao procurar a solução no site de buscas.

Porém todos os esquemas que achei não deram certo e algumas soluções envolve instalar aplicativos de terceiros e na maioria também é instalado várias dependências e isto para ver se "talvez funcione", mas fuçando em alguns comandos e verificando algumas informações no Arck Wiki, achei uma forma bem fácil de resolver sem instalar milhares de aplicativos no sistema somente para uma finalidade:

Fuçando, achei o comando "pulseaudio-ctl" e fui ver o que ele fazia, e é exatamente ele que me deu a idéia para a solução. Então o que vamos fazer é simplesmente um ou dois atalhos para o comando:

1) Abra o aplicativo "Todas as configurações" e clique na opção "Teclado", depois clique em "Adicionar";

2) A janela que abrir digite o comando a ser executado, para aumentar o volume é pulseaudio-ctl up e clique em OK;

3) A próxima janela é para definir o atalho, então segure o CTRL e aperte o botão de volume para aumentar.

Se tentar definir o atalho direto apenas com o botão de volume o atalho não irá funcionar, pois já está definido para o atalho de sistema padrão.
Pode fazer a mesma coisa para diminuir o volume, com o comando pulseaudio-ctl down

4) Edite o arquivo $HOME/.config/pulseaudio-ctl/config (se não existir, faça o comando "pulseaudio-ctl" no terminal para criar) e descomente todas as linhas. Os que tem no manjaro XFCE são estas:

Código: Selecionar todos
UPPER_THRESHOLD=150
NOTIFY=yes
BARCHART=yes


Se quiser que apareça a porcentagem do volume em vez da barra, deixe a opção BARCHART=yes comentada.

Em meu caso, meu som suporta até no máximo volume 153, não sei se é padrão para todos os que usam Linux e o pulseaudio, então deixei o valor igual no arquivo.

Configurando os atalhos de teclado desta forma, se quiser ultrapassar os 100% do volume, basta apertar CTRL+Botão de volume > ou <

Bônus:

Ao instalar o Manjaro XFCE, percebí que o atalho do Print Screen não funciona, então dá para aproveitar este esquema de configuração e definir um atalho para ele também, para chamar o comando xfce4-screenshooter
