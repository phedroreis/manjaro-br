Memória RAM e programas ativos: identificar o consumo

ID do artigo: 134
Adaptado por: Holmes
Adaptado em: 26 Mai 2016, 00:01
Descrição do artigo: Vamos conhecer o consumo da memória RAM pelos programas ativos!
Link externo para este artigo:
kb.html?a=134


Vamos identificar aplicativos que estejam rodando desnecessariamente e vazamentos de memória.

Primeiro baixe o script ps_mem.py:

Código: Selecionar todos
wget http://www.pixelbeat.org/scripts/ps_mem.py


Terminado o download, entre na pasta onde se encontra o script e rode o comando:

Código: Selecionar todos
sudo mv ps_mem.py /usr/local/sbin/


Depois disto rode ainda o comando:

Código: Selecionar todos
sudo chmod 755 /usr/local/sbin/ps_mem.py


Pronto, agora é só rodar o comando:

Código: Selecionar todos
sudo ps_mem.py


e você virá:

Código: Selecionar todos
[[email protected] ~]$ sudo ps_mem.py
