Acessar partição sem digitar senha

ID do artigo: 136
Adaptado por: ALisson Suassuna
Adaptado em: 28 Mai 2016, 12:45
Descrição do artigo: Artigo feito para soluciona o problemas de senhas na remoção de discos removíveis ou casso a eles.
Link externo para este artigo:
kb.html?a=136


Já implementado nas versões oficiais do Manjaro 0.8.4 nos sabores XFCE, Cinnamon e Openbox.


Um incômodo particular para muitos usuários é a necessidade de digitar uma senha para acessar outras partições presentes em seu disco rígido. Além disso, aqueles que instalou Openbox 0.8.3 também encontrou o problema adicional de ter de introduzir uma de suas senhas, a fim de ejetar a mídia removível, como cartões de memória USB e CD/DVDs. Isto é devido à política default / configurações de permissões implementadas por um programa chamado UDisks2, que em si é responsável por divisórias de montagem e mídia removível. Em essência, as permissões para montagem são definidas por padrão para o nível / administrador raiz única, ao invés do usuário. Felizmente, este problema pode ser facilmente resolvido através da criação de um arquivo especial chamado 99-manjaro.rules para substituir as permissões padrão, consequentemente resultando em mais nenhuma necessidade de digitar uma senha.

Vamos começa o processo:

Passo 1: Cria o arquivo de 99-manjaro.rules File

Você precisará primeiro abrir seu terminal , a fim de criar o arquivo de 99-manjaro.rules File. A sintaxe do comando para editar o mirrorlist é:

Código: Selecionar todos
sudo [Editor de texto] /etc/polkit-1/rules.d/99-manjaro.rules


Por exemplo, se você deseja editar o arquivo dentro do terminal usando o editor de texto > nano < ( um editor de texto padrão baseado em terminal), em seguida, digite:

Código: Selecionar todos
sudo nano /etc/polkit-1/rules.d/99-manjaro.rules


Caso contrário, se você tiver instalado a versão completa do Manjaro ( isto é, sem ser a versão NET-Edition), você pode achar que é mais fácil de usar um editor de texto como o Gedit ou Leafpad. Você também poderá usa esses editores, ficará a seu critério. Para usar Gedit digite o comando :

Código: Selecionar todos
gksu gedit /etc/polkit-1/rules.d/99-manjaro.rules


Passo 2: Editando o arquivo 99-manjaro.rules file

Abra o arquivo criado pelos comandos a cima. Uma vez que o arquivo em branco foi aberto, copie e cole o seguinte comando para substituir as permissões padrão utilizadas pelo UDisks2:

Código: Selecionar todos
polkit.addRule(function(action, subject) {
                if (action.id.indexOf("org.freedesktop.udisks2.") == 0 && subject.isInGroup("storage")) {
                        return polkit.Result.YES;
                }
        }
);


Depois de ter concluído o novo arquivo, salve as alterações e feche-a:

Se você estive usando o editor de texto nano. Para fecha utiliza essas instruções: Pressione CTRL e ' x ' para sair, 'y' para salvar e <enter> para terminar, ou Gedit ou Leafpad: Selecione a opção "salvar" e depois fechar a janela. Agora reinicie o sistema para que as alterações tenham efeito.
