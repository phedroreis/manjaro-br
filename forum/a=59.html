Como instalar o Radio Tray no Manjaro?

ID do artigo: 59
Adaptado por: lueneberg
Adaptado em: 02 Nov 2012, 15:18
Descrição do artigo: Aqui vamos saber como instalar o Radio Tray em nossa distribuição GNU/Linux.
Link externo para este artigo:
kb.html?a=59
Radio Tray é um aplicativo para gerenciamento de rádios online. Sua meta é ter uma interface mínima tornando-o muito simples de usar. Radio Tray não é um tocador de música cheio de recursos. Radio Tray é um software livre, licenciado sob a GPL.

Vamos ver como podemos instalar o Radio Tray no Manjaro. Na instalação do Radio Tray da problema no pacote python2-notify, que não está disponível. A solução é usar o arquivo PKGBUILD com o makepkg -s. Para isto você precisa baixar o arquivo, editar, tirar a referência do python2-notify, depois usar o makepkg -s para gerar e usar o pacman U para instalar. Vejamos a sequência de tudo isto:

1- abra o terminal e digite:

Código: Selecionar todos
wget https:/aur.archlinux.org/packages/ra/radiotray-hg/PKGBUILD


2- depois edite o arquivo

Código: Selecionar todos
gedit PKGBUILD


OBS.: na nona linha no final dela apague a referência ao python2-notify.

salve e de o comando:

Código: Selecionar todos
makepkg -s


OBS.: se tiver algum problema com o binário hg, instale o mercurial.

3- ele vai gerar o pacote radiotray-hg-487-1-any.pkg.tar.xz, após gerar dê o comando

Código: Selecionar todos
sudo pacman -U radiotray-hg-487-1-any.pkg.tar.xz


Pronto, o seu Radio Tray já está instalado e funcionando. Vejam umas imagens:
