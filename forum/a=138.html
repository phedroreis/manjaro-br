Melhorando a renderização das fontes

ID do artigo: 138
Adaptado por: PedroN
Adaptado em: 21 Jun 2016, 21:17
Descrição do artigo: O texto mostra como melhorar a renderização das fontes.
Link externo para este artigo:
kb.html?a=138


Como melhorar a renderização das fontes instaladas? Abaixo estão as instruções para isto.

1. Crie o arquivo /etc/fonts/local.conf.

Código: Selecionar todos
sudo nano /etc/fonts/local.conf


2. Cole o código seguinte no arquivo:

Código: Selecionar todos
<match target="font">
  <edit name="autohint" mode="assign">
    <bool>true</bool>
  </edit>
  <edit name="hinting" mode="assign">
    <bool>true</bool>
  </edit>
  <edit mode="assign" name="hintstyle">
    <const>hintslight</const>
  </edit>
  <edit mode="assign" name="lcdfilter">
   <const>lcddefault</const>
 </edit>
</match>


Após isso, salve o arquivo (Ctrl + O e Ctrl + X [Para sair]).

2. Crie um backup do arquivo ~/.Xresources

Código: Selecionar todos
cp ~/.Xresources ~/.Xresources.bak


Se o arquivo .Xresources não existir, você receberá esta mensagem de erro:

Código: Selecionar todos
cp: cannot stat ‘~/.Xresources’: No such file or directory


Então pule para a etapa 3.

3. Abra ou crie o arquivo ~/.Xresources com o editor de texto:

Código: Selecionar todos
nano ~/.Xresources


Delete o conteúdo (se presente algo) e cole este código no lugar:

Código: Selecionar todos
Xft.dpi: 96
Xft.antialias: true
Xft.hinting: true
Xft.rgba: rgb
Xft.autohint: false
Xft.hintstyle: hintslight
Xft.lcdfilter: lcddefault


Salve as alterações no arquivo (Ctrl + O e Ctrl + X [Para sair]).

4. Execute este comando no terminal:

Código: Selecionar todos
xrdb -merge ~/.Xresources


5. Tenha certeza que a suavização está ativada, e o Hiting está ajustado para Discreto nas Configurações de Sistema (Aparência).



Por fim, reinicie seu computador para que as configurações sejam aplicadas.
