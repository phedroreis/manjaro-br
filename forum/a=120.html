Comandos importantíssimos para serem usados no Manjaro

ID do artigo: 120
Adaptado por: Holmes
Adaptado em: 10 Mai 2015, 16:23
Descrição do artigo: O artigo abaixo vai mostar alguns comandos relevantes do pacman que podem ser usados na operacionalidade do Manjaro.
Link externo para este artigo:
kb.html?a=120


Wiki ArchLinux escreveu:O gerenciador de pacote pacman é uma das grandes vantagens do Arch Linux. Combina um simples pacote no formato binário, com um fácil uso de sistema de compilação. A meta do pacman é tornar o mais fácil possivel gerenciar pacotes, sejam eles dos oficiais repositórios Arch ou das próprias compilações do usuário. Pacman mantém o sistema atualizado, listas de pacotes de sincronização com o servidor mestre. Este modelo servidor/cliente também permite o usuário baixar/instalar pacotes com um simples comando, completo com todas as dependências requeridas. Pacman é escrito na linguagem de programação C e usa o formato de pacote .pkg.tar.xz.


Abaixo estão alguns comandos considerados significativos para serem usados no Manjaro ou distro baseadas no ArchLinux.

1. Atualizar o sistema

Código: Selecionar todos
sudo pacman -Syyuu


2. Otimizar os mirrors

Código: Selecionar todos
sudo pacman-mirrors -g
sudo pacman -Syy


3. Otimizar a base de dados

Código: Selecionar todos
sudo pacman-optimize


4. Limpar o cache

Código: Selecionar todos
paccache -ruk0
paccache -rk3


5. Pesquisar por pacotes

Código: Selecionar todos
pacman -Ss nome*


6. Informar sobre pacotes

Código: Selecionar todos
pacman -Si nome


7. Instalar pacotes

Código: Selecionar todos
sudo pacman -S nome


8. Detalhar pacotes instalados

Código: Selecionar todos
pacman -Qi nome


9. Remover pacotes

Código: Selecionar todos
sudo pacman -R nome


10. Remover pacotes + dependências

Código: Selecionar todos
sudo pacman -Rsn nome


11. Listar pacotes orfãos

Código: Selecionar todos
pacman -Qqdt


12. Remover pacotes orfãos

Código: Selecionar todos
sudo pacman -Rns $(pacman -Qqdt)


13. Listar pacotes por endereço localmente

Código: Selecionar todos
pacman -Ql nome


14. Listar pacotes por árvore de dependência

Código: Selecionar todos
pactree nome


15. Listar pacotes instalados dos repos oficiais

Código: Selecionar todos
pacman -Qen


16. Listar os pacotes (e suas dependências) instalados explicitamente com pacman ou yaourt, que ainda estejam em seu sistema.

Código: Selecionar todos
pacman -Qet


17. Corrigir erro db.lck

Código: Selecionar todos
sudo rm -f /var/lib/pacman/db.lck && sudo pacman-mirrors -g && sudo pacman –Syy


18. Atualizar o repo AUR

Código: Selecionar todos
yaourt -Syyuua


19. Pesquisar no repo AUR

Código: Selecionar todos
yaourt -Ss nome


20. Instalar pacotes do repo AUR

Código: Selecionar todos
yaourt -S nome


21. Remover pacotes do repo AUR

Código: Selecionar todos
yaourt -R nome


22. Remover pacotes + dependências do repo AUR

Código: Selecionar todos
yaourt -Rsn nome


23. Listar pacotes instalados do repo AUR

Código: Selecionar todos
pacman -Qem


Obs.: * nome do pacote.
